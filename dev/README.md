# To implement this project

first you have to create virtual environment

```bash
    python -m venv .venv
```

use virtual environment

 **windows**

```bash
    .\.venv\Scripts\activate
```

**macOS/Linux**

```bash
    source ./.venv/bin/activate
```

install dependency

```bash
    pip install -r requirements
```

# Build the project

On PowerShell:
```bash
./BuildScript.ps1
```
